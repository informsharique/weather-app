# How to run the project

 1. First clone the project using
 `$ git clone https://github.com/informsharique/react-native-coverflow.git`
 2. go to the root of the project and run

    `$ yarn install`
    OR
    `$ npm install`
 3. Run the project using

    `$ yarn android`
